# Backend System with Authentication

- Backend API.

## Tech Stack

- Node.js
- TypeScript
- Express.js
- Postgres
- Drizzle
- Docker

## Installation (Using Docker)

1. Run `docker compose up`
2. Use `Thunder Client\Postman`


## Installation (locally)

1. Clone the repository
2. Build and run the TypeScript code:
    ```zsh
    npm install
    npm run dev
    ```

## Directory Structure
